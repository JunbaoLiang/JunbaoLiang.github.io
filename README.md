## About the build

Static single-page site — no framework, no build step. Just `index.html`, `styles.css`, and a small `main.js` for the theme toggle and scroll behavior. Served from GitHub Pages.

Typography: Instrument Serif (display) · Geist (body) · JetBrains Mono (labels), all via Google Fonts. Dark theme by default with a light toggle that persists via `localStorage`.
